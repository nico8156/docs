# Frontend — Vue d’ensemble

L’app mobile est conçue pour rester fluide malgré :

- réseau instable
- latence
- offline temporaire
- reconnexions

## Principes

- UX first (réponses rapides, UI stable)
- Optimistic UI contrôlée
- Outbox client pour la résilience
- Reconciliation via read model + versioning

## Sections

- Navigation (tabs + stack)
- Map (chargement dynamique)
- Social (like/comment + temps réel)
- Ticket verify (workflow OCR)
