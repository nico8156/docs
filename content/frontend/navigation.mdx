# Navigation

## Objectif

Permettre une navigation cohérente tout en conservant la barre d’onglets visible.

## Approche

- Tabs pour les sections principales
- Stack pour les écrans “détails”
- Accès aux détails depuis plusieurs entrées (home, map, wishlist…)

## Ce que ça apporte

- UX fluide
- structure claire
- extensible (ajout d’écrans sans casser l’existant)
