# Ce que je peux apporter

Je suis disponible pour un **poste CDI** (ou mission) en **développement backend Java / Spring Boot** — Rennes ou remote.  
Mon objectif : livrer du code **simple, robuste, testé** et facile à faire évoluer.

---

## Backend Java (Spring Boot)

- APIs REST (Spring Boot), sécurité (Spring Security, OAuth2)
- Persistance SQL (JPA / PostGreSQL), gestion explicite des erreurs
- Conception orientée maintenabilité : séparation des responsabilités, refactoring, clean code
- Tests : unitaires → intégration → E2E (Testcontainers)

---

## Robustesse & architectures (quand le contexte le justifie)

- Découpage DDD / bounded contexts (pragmatique)
- Architecture hexagonale (ports & adapters)
- Flux asynchrones (Kafka), outbox transactionnelle
- Idempotence, retry/backoff, stratégie offline-first

---

## Mobile (React Native / Expo) — en complément

- Écrans et états synchronisés (Redux Toolkit)
- UX asynchrone (optimistic UI + reconciliation)
- Contraintes offline-first : queue persistée + retry

---

## Exemples de contributions

- Développement et stabilisation d’APIs backend (qualité, tests, lisibilité)
- Fiabilisation d’un pipeline asynchrone (events, retries, idempotence)
- Structuration progressive d’un codebase (séparation, tests, conventions)
- Mise en place de tests d’intégration reproductibles (Testcontainers)
