# Services

Je peux intervenir en **CDI** ou en **mission** (audit, refacto, delivery), avec un focus “production-ready”.

## Ce que j’apporte

### Architecture & robustesse
- Découpage DDD / bounded contexts
- Architecture hexagonale (ports & adapters)
- CQRS, Outbox transactionnelle, event-driven (Kafka)
- Idempotence, retry/backoff, stratégie offline-first

### Backend (Java/Spring)
- APIs REST, Spring Boot, Spring Security, OAuth2
- Workflows asynchrones et pipelines fiables
- Stratégies de tests : unit → intégration → E2E (Testcontainers)

### Mobile (React Native / Expo)
- UX asynchrone (optimistic UI + reconciliation)
- Outbox client (queue persistée, retry, idempotence)
- Cartographie, chargement dynamique, performance perçue
